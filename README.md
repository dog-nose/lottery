
# lottery for php

このプロジェクトは、PHPでランダム抽選を実装して、特定のインターフェイスを提供することで、ランダム抽出を簡単に実行することを目的とするプロジェクトです。

## requirements

- PHP 7.4 以上
- Composer
- PHPUnit (テスト実行用)

## development

```sh
# フォーマット
composer format
```

```sh
# リントチェック
composer lint
```

```sh
# テスト実行
composer test
```

```sh
# カバレッジレポート生成
composer coverage
```
